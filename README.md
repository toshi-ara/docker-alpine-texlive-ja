## Install

```bash
docker pull toshiara/alpine-texlive-ja
```

## Usage

```bash
$ docker run --rm -it -v $PWD:/workdir toshiara/alpine-texlive-ja
$ latexmk -C main.tex && latexmk main.tex && latexmk -c main.tex
```
